## What is PolyPay?

PolyPay is a privacy-preserving payroll platform that enables organizations, DAOs, and global teams to run payroll privately.

**Our Solution:** PolyPay uses zero-knowledge proofs to keep payment details confidential while maintaining on-chain security.
